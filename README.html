README – Profil‑Generator

Bitte starte die Anwendung über die Datei Launch-Profil-Generator.cmd. So öffnet sich die Seite korrekt im Browser und alles funktioniert zuverlässig.

1) Programm starten

  1. Doppelklicke auf Launch-Profil-Generator.cmd. Dein Browser öffnet sich automatisch.

2) Kurzen Text einfügen

  1. Im Feld „Kurzprofil (Freitext)“ den Text zum Kandidaten einfügen. Beispiel: Erfahrungen, Schwerpunkte, Ausbildung, Gehaltswunsch, Verfügbarkeit.
  2. Falls im Text „Profil 1234:“ vorkommt, wird die Nummer als Profil‑ID übernommen.

3) Kontaktdaten eintragen

  1. Dein Name, E‑Mail und Telefonnummer ausfüllen (Pflichtfelder).

4) Standort des Stellenangebots

  1. Trage den Ort ein (z. B. „München / Remote“). Dieser Ort dient als Ausgangspunkt; der Wohnort im Profil wird in plausibler Nähe dazu gewählt (nicht zwingend identisch).

5) OpenAI‑Schlüssel

  1. Trage deinen OpenAI API‑Schlüssel ein (beginnt meist mit sk-). Er wird nur lokal im Browser gespeichert. Wenn du keinen Schlüssel hast, wende dich an den System‑Administrator.
  2. Wähle bei Bedarf ein Modell.

6) Profil generieren

  1. Klicke auf „LLM generieren“. Kurz warten, dann zum Tab „Profil (Bearbeiten)“ wechseln.

7) Interne Kennung prüfen

  1. „Profil‑ID“, „Geburtsjahr“, „Wohnort“, „Familienstand (ledig)“, „Staatsangehörigkeit (deutsch)“ werden angezeigt.

8) Bearbeiten, Vorschau & PDF

  1. Bearbeite Inhalte im Tab „Profil (Bearbeiten)“.
  2. Nutze „Vollbild Vorschau“/„Eingabe anzeigen“ zum Umschalten.
  3. „Export als PDF“ erzeugt eine druckfertige Datei.
Tipp: Die Anwendung speichert deine Bearbeitungen lokal im Browser und lädt sie beim nächsten Start automatisch.

Fragen & Antworten (Q&A)

Was mache ich, wenn das Profil nicht passt? Wenn mir das Profil nicht gefällt?

Du hast zwei Möglichkeiten:

  1. Gezielt anpassen: Wechsle zum Tab „Profil (Bearbeiten)“ und ändere einzelne Felder (z. B. Berufsziel, Aufgaben, Ergebnisse, Wohnort usw.).
  2. Neu generieren: Klicke einfach erneut auf „LLM generieren“. Das Modell erstellt ein neues Profil. Oft hilft eine kleine Änderung im Freitext (z. B. Schwerpunkte konkreter nennen).

Welche Daten darf ich eingeben?

Bitte keine echten Arbeitgeber‑, Hochschul‑ oder Personennamen eintragen. Verwende nur allgemeine Beschreibungen (z. B. „Immobiliengesellschaft (KMU)“, „Region München“).

Der Wohnort wirkt unplausibel. Was tun?

Trage beim „Standort Stellenangebot“ einen Ort ein, der als Ausgangspunkt dient. Der Wohnort im Profil sollte in plausibler Nähe liegen. Alternativ im Tab „Profil (Bearbeiten)“ den Wohnort manuell anpassen.

Warum ist der Name immer „[Anonymisiert]“?

Zum Schutz der Privatsphäre. Das Profil ist für anonymisierte Vorabprüfungen gedacht. Reale Namen werden nicht verwendet.

Kann ich bestimmte Formulierungen anpassen?

Ja. Du kannst Texte in „Profil (Bearbeiten)“ frei anpassen (z. B. Aufgaben kürzen, Ergebnisse präzisieren, Tonalität ändern).

Wie bekomme ich einen OpenAI‑Schlüssel?

Wenn du noch keinen Schlüssel hast, wende dich bitte an den System‑Administrator.

Beim Export stimmt das Layout nicht ganz.

Nutze die PDF‑Exportfunktion im Tool. Das Ergebnis ist für A4 optimiert. Bei Darstellungsproblemen die Inhalte etwas kürzen oder „Vollbild Vorschau“ prüfen.
